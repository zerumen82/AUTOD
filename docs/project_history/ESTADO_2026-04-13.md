## 📝 Notas Técnicas

### FaceSwap Tab
- Métricas HTML: progreso %, archivos procesados, tiempo transcurrido, ETA, estado
- Procesamiento secuencial obligatorio para `temporal_smoothing=True`
- Soporte de imágenes: PNG, JPG, JPEG, WebP, BMP

### Image Editor Tab
- FLUX: 8 pasos, denoise 0.75, timeout 15s health, 30s history
- Qwen: 12 pasos, denoise 0.75, VAE 3D→2D via QwenVAELoader

### Arquitectura Qwen VAE
- `qwen_image_vae.safetensors` es un VAE 3D (WanVAE)
- Se convierte a 2D vía `QwenVAELoader` custom node
- Usa `comfy.utils.load_torch_file()` + `comfy.sd.VAE(sd=sd)`
